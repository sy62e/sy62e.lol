about

btrtv was a side project focused on making webchnl feel more usable on a tv. the idea was to keep the useful features from base webchnl and add more features made for tv.

highlights

  • remote-friendly controls
  • cleaner layout for big screens
  • custom styling and polish

what it aimed to do

the project was meant to be a stripped down version of the webchnl website that was less cluttered for tvs. putting the icons on the sides of the screen and adding controls for tv instead of using a mouse and keyboard. i also added a few features like channel programming and channel switching for 24/7 channels. it was never finished though

status

the project has been shutdown and i have no archives of it due to a loss of data.
